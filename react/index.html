Introduction

React is an open-source JavaScript library created by Facebook that is used to build UI components. This library is use as a base to develope mobile applications or single-page websites. It is used for handling the view layer for web and mobile apps. ReactJS allows us to create reusable UI components. It is currently one of the most popular JavaScript libraries and has a strong foundation and large community behind it.

According to React official website, following is the definition − "React is a library for building composable user interfaces. It encourages the creation of reusable UI components, which present data that changes over time. Lots of people use React as the V in MVC. React abstracts away the DOM from you, offering a simpler programming model and better performance. React can also render on the server using Node, and it can power native apps using React Native. React implements one-way reactive data flow, which reduces the boilerplate and is easier to reason about than traditional data binding."

Introducing JSX

The best way to use react is by using JSX which is an extened syntax of Javascript. JSX is like a template language with the power of Javascript it. Take a look at the following code:

const element = h1 Hello, world! /h1 ;

Now JSX is not required by react but it is a tool that is helpful as a visual when working with the inside UI Javascript.

Rendering Elements

The next concept you should learn is rendering elements. Elements are the building blocks of React because an element decribes what you want to see on the screen. For example:

div id="root" /div

In the example above you see that root is the name of the id this is what is managed by React DOM,Document Object Model. When you are working with a prexsisting website and would like to incoropate react you would need to pass the element throught two renderings.

const element = h1 Hello, world h1; ReactDOM.render(element, document.getElementById('root'));

The good thing about react is that you only have to call the render function once. Its important to know that react only updates when necessary.
